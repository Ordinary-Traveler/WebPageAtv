# Portfólio de Desenvolvimento Web - Unidade I

Este repositório reúne os projetos e atividades práticos desenvolvidos para a disciplina de Desenvolvimento Web I. O foco aqui é a aplicação de HTML5 semântico, CSS Grid e Flexbox para criar interfaces modernas e responsivas.

## 📂 Projetos em Destaque

Abaixo, você pode acessar a versão "live" de cada seção apresentada:

### 1. Página de Receitas (Atividade 01)
Uma landing page detalhando uma receita de Cupcake, focada em estruturação de texto e listas.
* **🔗 [Acessar Página de Receitas](https://ordinary-traveler.github.io/Cupcake-Sueli-Raquel_GuilhermeLadeia/)**

### 2. Portfólio Fullstack (Atividade 02)
A página que ajustamos juntos, contendo a seção Hero com os avatares, o grid de trabalhos e a seção de serviços.
* **🔗 [Acessar Portfólio Completo](https://ordinary-traveler.github.io/unidade_01_atividade_02/Portfolio-Sueli-Raquel/index.html)**

---

## 🛠️ Tecnologias de Destaque
* **Layout:** CSS Grid (3 colunas nos projetos) e Flexbox.
* **Estilo:** Dark Mode com acentos em cores vibrantes (amarelo, vermelho, roxo e verde).
* **Responsividade:** Adaptado para diferentes resoluções de tela.
