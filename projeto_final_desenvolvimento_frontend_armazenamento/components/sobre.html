Sobre o Projeto

O S.O.S Mulher nasceu da necessidade de criar um porto seguro digital para as mulheres de Nova Ipixuna - PA e região.

Nossa missão é combater a violência doméstica através da informação, do acolhimento e da facilitação do acesso aos canais de denúncia. Acreditamos que o conhecimento é o primeiro passo para a liberdade.

🛡️

Segurança: Ferramentas pensadas para proteger a usuária.

🤝

Apoio: Conexão direta com órgãos municipais e estaduais.

Espaço Seguro

Este site foi desenvolvido para ser leve, rápido e discreto. Cada funcionalidade, desde o mapa de locais até o botão de saída rápida, foi pensada para quem precisa de ajuda agora.

Importante: Este é um projeto informativo. Em caso de emergência, utilize sempre o botão de Denúncia.
